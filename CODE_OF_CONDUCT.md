# Code Of Conduct

## Our Pledge

We pledge to make participation in the RipDock Protocol community a respectful
and harassment-free experience for everyone.

## Expected Behavior

Participants are expected to:

- use welcoming and inclusive language
- respect differing viewpoints and experiences
- give and receive constructive feedback professionally
- focus criticism on ideas, protocol behavior, and implementation evidence
- accept responsibility for mistakes and correct them

## Unacceptable Behavior

Unacceptable behavior includes:

- harassment, intimidation, or discriminatory language
- personal attacks
- sexualized language or imagery
- trolling, insulting comments, or sustained disruption
- publishing private information without explicit permission
- other conduct that would reasonably be considered inappropriate in a
  professional setting

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, pull
requests, and other contributions that violate this Code of Conduct. Maintainers
may also temporarily or permanently ban participants for behavior they consider
inappropriate, threatening, offensive, or harmful.

## Reporting

Report Code of Conduct concerns privately by emailing:

support@ripdock.com

Security vulnerabilities should be reported according to `SECURITY.md` instead.

## Scope

This Code of Conduct applies within project spaces and when an individual is
representing the project in public spaces.
